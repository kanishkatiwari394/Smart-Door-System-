# Smart Door System Using Arduino Uno

## 📌 Project Overview
This project implements a Smart Door System using an Arduino Uno, PIR Motion Sensor, Servo Motor, and LED. When motion is detected by the PIR sensor, the servo motor opens the door and the LED turns ON. When no motion is detected, the door closes and the LED turns OFF.

## 🛠 Components Used
- Arduino Uno
- PIR Motion Sensor
- Servo Motor
- LED
- 220Ω Resistor
- Jumper Wires


## ⚙ Working Principle
1. PIR sensor continuously detects motion.
2. When motion is detected:
   - LED turns ON.
   - Servo rotates to 90° (door opens).
   - Serial Monitor displays **"Motion Detected"**.
3. When no motion is detected:
   - LED turns OFF.
   - Servo returns to 0° (door closes).
   - Serial Monitor displays **"No Motion"**.

## 💻 Arduino Code Features
- Motion detection using PIR sensor.
- Automatic door opening and closing.
- LED status indication.
- Serial Monitor output for monitoring system status.

## 🚀 Applications
- Smart Home Automation
- Automatic Entry Doors
- Security Systems
- Office Access Control
- Motion-Based Door Systems
